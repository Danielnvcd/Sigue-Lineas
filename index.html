Robot Seguidor de Línea

Materiales Necesarios

  • Arduino Nano
  • Driver de Motor TB6612FNG
  • Regleta de Sensores QTR8A
  • 2 micro reductores 1:10 con sus Brackets
  • Ruedas (tipo rodillo o normales)
  • Cables, estaño, cautín, etc.
  • Batería de 7.4v (Litio o Lipo de 2 celdas)

Conexiones y Construcción

Realiza las conexiones como indica el esquema y monta todo sobre un chasis ligero (como de baquelita).

Código de Arduino

El código usa sensores QTR y un algoritmo PID para seguir la línea.
